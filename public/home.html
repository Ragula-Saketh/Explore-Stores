Explore stores

Explore Stores is a dynamic platform
designed to manage a list of stores
with full CRUD functionality.

Website Features

Add: Add a new store to the list by filling out the form with
relevant details.

Delete: Remove any store from the list using the delete option
on the store card.

Update: Edit the details of an existing store by clicking on the
edit button.

View: View the details of any store by selecting the edit
option (without making changes) or clicking on image.
